Проект: Чат-бот для агентства недвижимости АН «МЭтр»
Этот репозиторий содержит дизайн-спецификацию и логику диалогового агента (чат-бота), разработанного для агентства недвижимости АН «МЭтр» из города Самара. Проект фокусируется на автоматизации первичного взаимодействия с потенциальными клиентами с целью увеличения числа назначенных просмотров квартир.

Описание
Чат-бот выступает в роли виртуального помощника-риэлтора. Его основная задача — не просто отвечать на вопросы, а вести клиента по структурированному сценарию к конкретному результату: записи на просмотр объекта недвижимости или согласию на обратный звонок от менеджера.

Бот основан на реальных бизнес-процессах и уникальных преимуществах агентства, таких как работа со сложными ипотечными случаями и оплата услуг по факту достижения результата.

Ключевые особенности
Целенаправленная логика диалога: сценарий разделен на 6 четких шагов — от приветствия до фиксации действия.
Деликатная работа с возражениями: встроенные ответы на типичные фразы вроде «просто смотрю» или «подумаю».
Соблюдение этических норм: в сценарии запрещены давление, гарантии и юридические консультации.
Фокус на конверсию: каждый ответ в диалоге приближает пользователя к главной цели — просмотру.
Бизнес-контекст
Проект создан для агентства недвижимости АН «МЭтр», расположенного по адресу: г. Самара, ул. Куйбышева, 128. Агентство специализируется на:

Подборе квартир на вторичном рынке и в новостройках.
Оформлении ипотеки, включая сложные случаи (от 2,7%).
Рефинансировании и кредитовании.
Сопровождении сделок «под ключ» с электронной регистрацией.
Работе с материнским капиталом и госсертификатами.
Агентство имеет положительную репутацию и высокий рейтинг (4.3 на Яндекс Картах) среди клиентов в Самаре 1.

Структура проекта
Проект представляет собой спецификацию, которую можно легко интегрировать в любую популярную платформу для создания чат-ботов (Telegram Bot API, ManyChat, Landbot и др.).

Как использовать
Склонируйте репозиторий.
Изучите файл bot_prompt_spec.md, содержащий полную инструкцию для настройки бота.
Адаптируйте спецификацию под синтаксис и возможности вашей целевой платформы.
Подключите бота к каналам коммуникации агентства (сайт, мессенджеры).
